Description by Astar Wallet

Adds support for Astar, an interoperable blockchain platform for Polkadot and Ethereum ecosystems supporting both Wasm and EVM smart contracts. After installing the Snap, visit the Astar Portal at portal.astar.network to connect. Make sure to select 'Astar Snap' under the Native Accounts options.
